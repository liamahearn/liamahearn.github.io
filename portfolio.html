Tripfol.io

My team and I built Tripfol.io, a day-trip generator that uses routing algorithms, live real-world traffic information, and location data via INRIX and Yelp APIs. I served as the sole UI Designer and Frontend Programmer on the team, drafting up layouts in Figma, and using Flutter to build the app while my teammates focused on the extensive backend that the project required. Check out the submission on Devpost here!

Project Pineapple

My team and I created Project Pineapple, an iPhone app that offers an extra layer of safety by scheduling check-ins before a night out, contacting friends if needed. I served as the UI designer and as a frontend programmer for the team, where I designed the UI in Figma, using Swift to implement them into the project. Check out the Devpost submission here!

FastThru

My first foray into UI design, my team and I built FastThru, a web app that takes INRIX API traffic/location data and helps users find the shortest drive-thru lines nearby. I acted as head UI / Graphic Designer for the team, where I created sketches, interface designs, and icons in Figma, implementing them with HTML/CSS. Check out the final product here!
